# cistercian

Commandline tool to generate [Cistercian numerals].

## Installation

```bash
go get github.com/rhardih/cistercian
```

## Example usage

```
$ ./cistercian 7323
    ╷
    │╲
    │ ╲
    │  ╲
────┤
    │
    │
    │
╷   │
│   │  ╱
│   │ ╱
│   │╱
└───┘
```

[Cistercian numerals]: https://en.wikipedia.org/wiki/Cistercian_numerals
